# ZJU_Summer_School DV Project

## How to run
构建项目时请使用nodejs 18版本及以上，使用pnpm作为包管理工具。
- requirements
  - nodejs@18 https://nodejs.org/en/download
  - pnpm https://pnpm.io/installation

- setup project
  ```shell
  pnpm install
  ```

- run dev server
  ```shell
  pnpm run dev
  ```

- open webpage http://localhost:5173/
